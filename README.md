# Job Tracker

## Run locally

Use Node.js 20.19+ or 22.12+.

Start the backend:

```bash
cd back
npm install
npm run dev
```

The API runs at `http://localhost:4000`.

In a second terminal, start the frontend:

```bash
cd front
npm install
npm run dev
```

Open the URL printed by Vite, usually `http://localhost:5173`.
